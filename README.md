# Modrinth CLI

This is a work in-progress implementation of Modrinth command-line interface in .NET.

## Features

- [x] Search (`search`)
- [x] Pinging (`ping`)
- [ ] Get project details (`get`)
- [ ] Download project file (`download`)

## Usage

```sh
./Modrinth.Console [options]

# See help using:
./Modrinth.Console help
```
